# Population Changes

This example uses U.S. Census percent-change data for the 50 states from 1910 to 2020.

Data source: `population-change-data-table.xlsx`
Source:
`U.S. Department of Commerce`
`U.S. Census Bureau`
`Change in Resident Population of the 50 States, the District of Columbia, and Puerto Rico: 1910 to 2020`

- Most changed overall: `Nevada` with about `46.9%` average absolute population change per decade
- Least changed overall: `Iowa` with about `3.9%` average absolute population change per decade
- Largest single-decade change: `Washington` with `120.4%` in `1910`
